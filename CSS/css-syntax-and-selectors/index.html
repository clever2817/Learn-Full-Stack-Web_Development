CSS Syntax and Selectors

There are three ways to add CSS. These are:

  • Inline CSS (It has the most precedence but not recommended to use.)
  • Internal CSS (It has the second most precedence but still not recommended to use.)
  • External CSS(It has the least precedence but it is highly recommended to use because of its cleanliness and reuse ability.)
